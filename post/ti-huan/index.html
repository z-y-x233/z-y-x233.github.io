替换

发布于 2020-11-23 | 分类于 周赛 | 2分钟 | 427字数

题目描述

给定一个长度为nnn的整数序列，对这个序列进行qqq次操作
每次操作给出两个数a,ba, ba,b，代表把序列中的aaa全部替换为bbb

因为我们需要求的答案为数列和，而对于每一次的操作qqq,会把整个数组和和sumsumsum减去aaa的数量乘aaa再加上aaa的数量乘bbb
所以我们只需要求得他们的差值并且修改他们就行
而因为数据范围为1e51e51e5级别的数，所以直接暴力修改替换肯定会超时
这时我们可以注意到a,ba,ba,b的范围也为1e51e51e5，那么我们可以利用桶排的思想
建立一个nnn大于等于1e51e51e5数组sns_nsn​,把a,ba,ba,b直接做为数组下标
用sxs_xsx​表示xxx这个数出现了多少次，在读入中每次读入一个xxx，就把sxs_xsx​加一

那么这样存储数据后，对于每一次操作qqq我们可以令sum=sum−sa∗a+sa∗bsum =sum - s_a * a + s_a * bsum=sum−sa​∗a+sa​∗b
然后修改a,ba,ba,b的数量,sb+=sa,sa=0s_b += s_a, s_a = 0sb​+=sa​,sa​=0
这样对于每一次操作qqq，只需要花费O(1)O(1)O(1)的时间，对于每次操作后输出sumsumsum即可
代码


#include<bits/stdc++.h>

using namespace std;

const int N = 100005;
int e[N];
long long sum = 0;

int main(void) {
	int  n, q, x, a, b;
	cin >> n >> q;
	for (int i = 0; i < n; ++i) {
		scanf("%d", &x);
		e[x]++;
		sum += x;
	}
	while (q--) {
		scanf("%d%d", &a, &b);
		if (a == b) {
			printf("%lld\n", sum);
			continue;
		}
		sum -= e[a] * a;
		e[b] += e[a];
		sum += e[a] * b;
		e[a] = 0;
		printf("%lld\n", sum);
	}
}
