HHKB Programming Contest 2020 D Square

2020-12-09 · 5 min read

题目描述

原题链接 D Squares

给定三个正方形N，A，B。

边长分别为n,a,bn,a,bn,a,b, 求有多少种方式把A，B放入N中且A，B不重合。

方案数对1e9+71e9 + 71e9+7 取模

数据范围

$1\le n \le 1e9 $

1≤a,b≤n1 \le a,b \le n1≤a,b≤n

分析

对于直接在二维平面上求满足条件的情况十分困难~~(反正我不会)~~

所以我可以把正方形投影到xxx轴和yyy轴上，转换为对一维坐标的分析

对于重合的两个正方形而言，他们的xxx和和和yyy轴上的投影必然同时重合，否则只要一个坐标轴上的边不重合，那么两个正方形必然不重合

所以我们可以分成几组组情况讨论

  • xxx轴不重合，yyy轴坐标随便放

  • yyy轴不重合，xxx轴坐标随便放

  • xxx轴不重合，yyy轴不重合

答案即为前两种情况的和减去最后一种情况，因为两个坐标轴都不重合的情况在前两种情况中都计算了一次

第一种情况：xxx轴不重合，yyy轴坐标随便放

因为两个正方形的长度是固定的，所以我们只考虑左端点来代表正方形。同时考虑两个正方形的左右是复杂的，但是哦们可以注意到，左右两种情况其实是对称的，所以我们只需要讨论一种情况，最后把结果乘以2就行了，我们假设A在B的左边。

当A在000点的时候，B有n−a−b+1n - a - b + 1n−a−b+1 种放法，每当A点向右移动一位，B的放发减一，当A在点n−a−bn - a - bn−a−b的时候，B只剩一种放法，放在点n−bn - bn−b

我们令d=n−a−bd = n - a - bd=n−a−b

所以对于xxx轴不重合的情况下，在xxx轴上两点的所有放置方法数量为

sx=∑i=id+1i=(d+1)(d+2)2s_x = \sum _{i = i}^{d + 1} i = \frac{(d +1)(d + 2)}{2} sx​=i=i∑d+1​i=2(d+1)(d+2)​

而此时yyy轴上因为可以任意放置，所以yyy轴上的放置数量为sy=(n−a+1)(n−b+1)s_y = (n - a + 1)(n - b + 1)sy​=(n−a+1)(n−b+1)

因此根据乘法原理，总的放置数量为s=sx×sys = s_x \times s_ys=sx​×sy​

第二种情况：yyy轴不重合，xxx轴坐标随便放

因为是正方形，所以这种情况和第一种是完全等价的，因此该种情况数量也为sss

第三种情况：xxx轴不重合，yyy轴不重合

对于任意一个坐标轴，不重合的放置方法数都为

r=∑i=id+1i=(d+1)(d+2)2r = \sum _{i = i}^{d + 1} i = \frac{(d +1)(d + 2)}{2} r=i=i∑d+1​i=2(d+1)(d+2)​

所以总的放置数量即为r2r^2r2

最后的答案即为2s−r22s - r^22s−r2，当a+b>na + b > na+b>n的时候需要特判一下答案等于000

#pragma comment(linker,"/STACK:1024000000,1024000000") 

#include<iostream>
#include<algorithm>
#include<vector>
#include<cmath>
#include<unordered_map>
using namespace std;

#define max(a,b) (((a) > (b)) ? (a) : (b))
#define min(a,b) (((a) < (b)) ? (a) : (b))

const double pi = acos(-1);
const int N = 2e5 + 100;
const int mod = 1e9 + 7;
typedef long long ll;
template<typename T> 
inline void read(T &x) {
	x = 0;
	int  w = 1;
	char ch = 0;
	while (ch < '0' || ch > '9') {
		if (ch == '-') w = -1;
		ch = getchar();
	}
	while (ch >= '0' && ch <= '9') {
		x = (x << 1) + (x << 3) + (ch - 48);
		ch = getchar();
	}
	x *= w;
}


void qwq() {
	ll n, a, b;
	cin >> n >> a >> b;
	if (a + b > n) {
		cout << "0\n";
		return;
	}
	ll s = (n - a + 1 ) * (n - b + 1) % mod;
	ll d = (n - b - a) % mod;
	ll res1 = (((d + 1) * (d + 2) % mod) * s % mod) * 2 % mod;
	ll res2 = (((d + 1) * (d + 2)) % mod) * ((d + 1) * (d + 2) % mod) % mod;
	cout << (res1 - res2 + mod) % mod << endl;

}


 int main() {
	int _ = 1;
	read(_);
	while (_--) qwq();
	return 0;
}
